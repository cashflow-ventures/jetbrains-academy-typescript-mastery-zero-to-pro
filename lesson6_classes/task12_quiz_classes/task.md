# Quiz: Classes

Test your understanding of TypeScript classes — access modifiers, abstract classes, interfaces,
and structural typing.

## Select ALL correct statements about TypeScript classes:

Think carefully about each option. Consider what `private` vs `protected` means, whether
abstract classes can be instantiated, how `implements` differs from `extends`, and what
happens to interfaces at runtime.

<div class="hint">
Remember: `private` restricts access to the declaring class only — not even subclasses can
see it. `protected` opens access to subclasses. Abstract classes exist to be extended, not
instantiated. Interfaces are a compile-time concept — they're erased from the JavaScript output.
</div>
