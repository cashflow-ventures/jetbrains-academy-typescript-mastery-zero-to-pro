# Quiz: Basic Types

Test your understanding of the types covered in this lesson.

## Which of the following statements about TypeScript types is correct?

Consider what you've learned about primitive types, arrays, tuples, and the special types
`any`, `unknown`, and `never`. Think carefully about how each type behaves and what
TypeScript allows or prevents.

<div class="hint">
Think about the key difference between `unknown` and `any`: one requires narrowing before
use, the other doesn't. Also consider what makes a tuple different from a regular array —
it's about fixed length and per-position types.
</div>
