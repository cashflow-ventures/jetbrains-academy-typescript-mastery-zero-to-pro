# Quiz: Design Patterns

Choose the correct answer about TypeScript design patterns.

## Which pattern is described by: "Define a family of interchangeable algorithms, encapsulate each one behind a common interface, and make them swappable at runtime"?

Think about which pattern focuses specifically on making algorithms pluggable and interchangeable without modifying the code that uses them.

<div class="hint">
Consider what each pattern's primary purpose is:
- One pattern builds complex objects step by step
- One pattern notifies subscribers when events occur
- One pattern abstracts data storage behind a generic interface
- One pattern provides dependencies from the outside
- One pattern makes algorithms interchangeable via a shared interface
</div>
