# Quiz: Enums

Choose the correct answer about TypeScript enums.

## Which statement about reverse mapping, const enums, and union literal types is true?

Consider how each enum variant compiles to JavaScript and what features are available
at runtime versus compile time.

<div class="hint">
Think about what TypeScript emits for each kind of enum. Numeric enums produce a
two-way mapping object. String enums produce a one-way object. Const enums produce
no object at all — their values are inlined. Union literal types are erased entirely.
</div>
